# BSPWM 

## Configuration for navigating windows on a AZERTY keyboard

```
# Switch to desktop 1
super + ampersand
    bspc desktop -f 1

# Move focused window to desktop 1 and follow
super + shift + ampersand
    bspc node -d 1 --follow
```
